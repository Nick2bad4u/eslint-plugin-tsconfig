# 🟣 All

Use when you explicitly want every plugin rule, including experimental coverage.

## Config key

```ts
tsconfig.configs.all
```

## Flat Config example

```ts
import tsconfig from "eslint-plugin-tsconfig";

export default [tsconfig.configs.all];
```
