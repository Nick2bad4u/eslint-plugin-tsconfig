# 🟠 Recommended (type-checked)

> **Note:** `eslint-plugin-tsconfig` does not have a `recommended-type-checked` preset.
> All rules use `jsonc-eslint-parser` and do not require type information.
> Use [`tsconfig.configs.recommended`](./recommended.md) or [`tsconfig.configs.strict`](./strict.md).
